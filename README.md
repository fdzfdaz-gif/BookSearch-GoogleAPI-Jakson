# GoogleBooksAPIを使った本を検索するCLI

## 主な機能
- キーワードを使った本の検索。
- GoogleBooksAPIとの連携。
- SSONデータのJavaへのマッピング。

## 使用方法
- Mavenで依存関係をインストールします。
- `Main` クラスを実行し、コンソールに検索したいキーワードを入力。

# CLI for Searching Books Using the Google Books API

## Key Features
- Search for books using keywords.
- Integration with the Google Books API.
- Mapping JSON data to Java.

## How to Use
- Install dependencies using Maven.
- Run the `Main` class and enter the keyword you want to search for in the console.
